Chamfer ships as framework-agnostic CSS plus optional enhancements. Follow the steps below to install the packages, pull in the baseline styles, and render your first button.

## Overview

### Requirements

- Node.js 18.18+ (20.x recommended)
- A package manager (`pnpm`, `npm`, or `yarn`)
- A bundler that understands ESM imports (Vite, Next.js, Remix, Astro, etc.)

## 1. Install the packages

```bash
# pnpm
pnpm add @chamfer/core @chamfer/behavior

# npm
npm install @chamfer/core @chamfer/behavior

# yarn
yarn add @chamfer/core @chamfer/behavior
```

> Need Tailwind utilities? Add `@chamfer/tailwind` to the same install command.

## 2. Import the baseline CSS

### HTML entry point

```html
<!-- app.html -->
<link rel="stylesheet" href="/node_modules/@chamfer/core/dist/css/index.css" />
```

### Bundled setup

```ts
// main.ts
import "@chamfer/core/css";
```

These imports expose the generated token layers, resets, and component styles. No additional PostCSS configuration is required.

## 3. Render your first button

<DemoBlock demo={buttonBaselineExample} component={ButtonBaselineExample} />

```html
<button class="ch-button ch-button--md ch-button--solid ch-button--primary" data-ch-component="button">
  <span class="ch-button__label">Save changes</span>
</button>
```

- `data-ch-component="button"` provides a stable hook for styling and tests.
- Size, emphasis, and tone are controlled through BEM-style modifiers (for example `ch-button--md`, `ch-button--solid`, `ch-button--primary`).

## 4. Add ripple behavior (optional)

```ts
import { enhanceButton } from "@chamfer/behavior";

document.querySelectorAll('[data-ch-component="button"]').forEach(element => {
  const enhancement = enhanceButton(element as HTMLButtonElement);
  element.addEventListener("disconnected", enhancement.destroy, { once: true });
});
```

React example:

```tsx
import "@chamfer/core/css";
import { useEffect, useRef } from "react";
import { enhanceButton } from "@chamfer/behavior";

export function SaveButton() {
  const ref = useRef<HTMLButtonElement>(null);

  useEffect(() => {
    if (!ref.current) return;
    const enhancement = enhanceButton(ref.current);
    return enhancement.destroy;
  }, []);

  return (
    <button
      ref={ref}
      className="ch-button ch-button--md ch-button--solid ch-button--primary"
      data-ch-component="button"
    >
      <span className="ch-button__label">Save changes</span>
    </button>
  );
}
```

## 5. Tailwind integration (optional)

```ts
// tailwind.config.ts
import { chamferTailwindPreset } from "@chamfer/tailwind";

export default {
  presets: [chamferTailwindPreset]
};
```

## 6. Explore the playground

```bash
pnpm --filter @chamfer/playground-html dev
```

The HTML playground provides live previews of tokens, surfaces, and components. If you edit token sources, rebuild before launching: `pnpm --filter @chamfer/tokens build` followed by `pnpm --filter @chamfer/core build`.
